# Study Program Manager

This program is a simple **study manager** that lets you add your subjects along with their study time and the day of the week.

## Versions

- **Beginner**  
  A simple and easy-to-understand version with repetitive code, suitable for beginners or those learning Python.

- **Pro**  
  A clean and optimized version with no repetitive code, more concise and scalable.

## Features

- Add new subjects with study time and day
- View your study program
- Easy to extend and modify
